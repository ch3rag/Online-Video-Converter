Requirements
-----------------------------
1. node.js
2. mongoDb
3. ffmpeg (Add PATH for ffmpeg.exe & ffprobe.exe)


For Client
-----------------------------

1. Install node.js.
2. Open terminal in "video-converter-client" folder.
3. Run "npm install" to install all dependencies.
4. 
  a) Run "npm run serve" to start web application.
  b) Run "npm run electron:serve" to start native application.



For Server
-----------------------------

1. Install node.js.
2. Open terminal in "video-converter-server" folder.
3. Run "npm install" to install all dependencies.
4. Run "npm run dev" to start development server.
